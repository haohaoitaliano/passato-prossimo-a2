# Il passato prossimo

## 项目说明

这是一个意大利语 A2 近过去时在线测试。

学生先完成 16 道近过去时填空题，提交后自动批改并显示分数、正确答案，然后解锁规则总结练习。

## 文件说明

- `index.html`：测试网页
- `.nojekyll`：保证 GitHub Pages 正常加载

## GitHub Pages 开启方式

进入：

`Settings → Pages`

在：

`Build and deployment`

选择：

`Deploy from a branch`

然后选择：

- Branch：`main`
- Folder：`/root`

保存后等待部署完成。

最终网址通常为：

`https://haohaoitaliano.github.io/passato-prossimo-a2/`
